# TalentLink AI | The Intelligent Workforce Hub

TalentLink AI is an enterprise-grade professional network for high-performance experts. It leverages real-time professional data synchronization and matching intelligence to bridge the gap between elite talent and visionary projects.

## Core Features

- **Professional Identity Core**: 100% data-driven professional profiles persisted in Firestore.
- **Intelligence Hub**: Real-time project matching and synchronization.
- **Gig Ideator**: High-impact opportunity analysis based on technical capabilities.
- **Neural Proposal Engine**: Persuasive pitch synthesis tailored to specific project briefs.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Firestore
- **Auth**: Firebase Authentication
- **AI Engine**: Genkit (Google Gemini 2.5 Flash)
- **Styling**: Tailwind CSS & ShadCN UI

## Getting Started

1.  **Firebase Setup**: Enable Email/Password authentication in your Firebase Console.
2.  **Environment**: Populate `src/firebase/config.ts` with your specific project credentials.
3.  **Deployment**: Optimized for Firebase App Hosting.
